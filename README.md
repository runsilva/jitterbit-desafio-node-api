 📦 Orders API

API REST para gerenciamento de pedidos, desenvolvida com **Node.js**, **Express** e **PostgreSQL**

## Instalação
npm install

## Configuração
Copie o .env.example e preencha as variáveis.

## Execução
npm run dev

## Endpoints
- POST   /order
- GET    /order/list
- GET    /order/:orderId
- PUT    /order/:orderId
- DELETE /order/:orderId
